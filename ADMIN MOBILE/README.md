# Interface Admin vs Interface Mobile

## Interface Admin

- Gestion des utilisateurs
- Accès aux statistiques avancées
- Paramétrage global de l’application
- Gestion des droits et rôles
- Visualisation et exportation des rapports

## Interface Mobile

- Accès simplifié pour l’utilisateur final
- Fonctionnalités principales accessibles rapidement
- Notifications en temps réel
- Saisie et consultation des données essentielles
- Interface adaptée aux écrans mobiles
