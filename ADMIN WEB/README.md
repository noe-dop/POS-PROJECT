# Interface Admin Purement Web

Ce projet propose une interface d'administration entièrement web pour la gestion de POS Globale Remote.

## Fonctionnalités principales

- Tableau de bord interactif
- Gestion des utilisateurs et des rôles
- Visualisation des transactions
- Paramétrage des points de vente
- Statistiques et rapports

## Technologies utilisées

- Frontend : React.js / Vue.js / Angular (à préciser)
- Backend : Node.js / Express / API REST (à préciser)
- Base de données : MongoDB / PostgreSQL (à préciser)

## Installation

```bash
git clone <repository-url>
cd <nom-du-projet>
npm install
npm start
```
