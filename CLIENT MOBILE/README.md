# Interface Client Consommateur Final Mobile

Cette application mobile est conçue pour offrir une expérience utilisateur optimale sur tous types d'appareils mobiles, y compris :

- **iOS** (iPhone, iPad)
- **Android** (smartphones, tablettes)

## Fonctionnalités principales

- Interface responsive adaptée à toutes les tailles d'écran
- Navigation intuitive pour les utilisateurs finaux
- Compatibilité multi-appareils (téléphones et tablettes)
- Support des dernières versions iOS et Android

## Objectif

Permettre aux clients finaux d'accéder facilement aux services de la plateforme, où qu'ils soient et quel que soit leur appareil mobile.
